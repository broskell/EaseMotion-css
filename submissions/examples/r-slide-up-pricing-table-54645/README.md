# CSS Slide-Up Pricing Table

A modern **CSS Slide-Up Pricing Table** built using pure HTML5 and CSS3 for creative portfolio layouts. The component features smooth slide-up entrance animations, subtle hover interactions, responsive layouts, and accessibility support.

---

## ✨ Features

- Pure HTML5 & CSS3
- Smooth slide-up entrance animation
- Hover lift interaction
- Responsive pricing card grid
- Featured pricing plan with badge
- CSS Custom Properties for easy customization
- Modern gradient buttons
- Mobile-first responsive design
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the files into your EaseMotion CSS project.
2. Link the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in your browser.

## 🎨 CSS Custom Properties

The component can be customized using the following variables:

```css
:root{
    --primary;
    --secondary;
    --background;
    --card;
    --text;
    --muted;
    --radius;
    --shadow;
    --duration;
}
```
