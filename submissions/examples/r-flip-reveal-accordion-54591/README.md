# CSS Flip-Reveal Accordion

A modern **Flip-Reveal Accordion** built using pure HTML5 and CSS3 for creative portfolio layouts. It leverages the native `<details>` and `<summary>` elements with a smooth 3D flip reveal animation, responsive design, and accessibility support—all without JavaScript.

---

## ✨ Features

- Pure HTML5 & CSS3
- Native `<details>` / `<summary>` accordion
- 3D flip-reveal opening animation
- Smooth expand and collapse transitions
- Responsive layout
- CSS Custom Properties
- Accessible semantic HTML
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in any modern browser.
