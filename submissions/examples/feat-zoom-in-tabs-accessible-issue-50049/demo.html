Contrast that holds up

Every text and border color pair in this component meets WCAG AA contrast (4.5:1 for body text) against its background — including the "dim" secondary text, which is often the first thing to fail in components like this.

Keyboard-first by construction

Tabs are native radio inputs, so keyboard support isn't bolted on — arrow keys move between tabs and Tab/Shift+Tab enter and leave the group, exactly as users already expect from radio groups.

  • Tab: focus enters the group on the checked tab
  • Arrow keys: move selection between tabs
  • Focus ring uses a color distinct from the selected-state color

Motion that respects the setting

The zoom-in is short and subtle by default (200ms, 4% scale change) rather than a showy effect, and it's removed entirely — not just shortened — for anyone with prefers-reduced-motion: reduce set.
