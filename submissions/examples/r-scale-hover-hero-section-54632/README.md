# CSS Scale-Hover Hero Section

A modern **Scale-Hover Hero Section** built using pure HTML5 and CSS3 for creative portfolio layouts. This component features layered portfolio cards with smooth scale-on-hover interactions, responsive design, and accessibility support without requiring JavaScript.

---

## ✨ Features

- Pure HTML5 & CSS3
- Scale-hover animation for preview cards
- Responsive two-column hero layout
- Modern gradient CTA buttons
- CSS Custom Properties for easy customization
- Lightweight and performant animations
- Keyboard-friendly interactive elements
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in any modern browser.
